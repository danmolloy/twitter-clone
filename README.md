# twitter-clone

## Features
* Post your own tweets
* Comment, like and retweet posts
* Follow other users
* Send direct messages to other users
* View user profiles
* Change your profile picture & blurb
* Encrypted authentication

## Stack
* React
* TypeScript
* Tailwind
* React-Router 
* Jest
* GraphQL (Apollo Client & Apollo Server) 
* Prisma with SQLite
